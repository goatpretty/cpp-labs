# 综合训练项目说明

## 1. 源码编码要求

- 本项目的 `.cpp` 源文件采用 **GBK 编码** 保存。
- 请使用支持编码选择的编辑器打开/保存，避免被自动转为 UTF-8 导致中文输出乱码或编译警告。

> 若你使用 VS Code：请在右下角编码处选择 **GBK** 打开，并以 **GBK** 重新保存。

---

## 2. 综合训练一：文件读写相关说明

### 文件说明

- `input.txt`：输入数据文件  
- `output.csv`：程序输出结果文件（运行后生成/覆盖）

### input.txt 的要求 **（不要删除）**

- `input.txt` 内包含 **输入格式说明** 与 **输入样例数据**，程序运行时会从该文件读取学生信息。
- 请勿删除 `input.txt`，也不要随意改名或移动文件位置，否则程序会提示无法打开文件。

### 输出文件 output.csv

- 程序在选择导出后，会生成 `output.csv`。
- CSV 以逗号分隔，包含表头，便于使用 Excel/WPS 直接打开查看与打印。

### 运行注意事项

- 请保证 `input.txt` 与可执行程序（或工程运行目录）在同一目录下。
- 程序会在控制台提示输入：学生人数、课程满分等信息，请按提示输入数值。
